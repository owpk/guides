Общие команды
fdisk -l - список всех устройств
fdisk /dev/sda - заходим в утилиту fdisk диска sda. Далее в интерфейсе fdisk n = создать новый раздел (+1G), d = удалить.

О скриптах Bash подробнее здесь
for ((i = 0; i < 1200; i++)); do touch ./dir/file_$i; done - создать 1200 файлов с именем file_$i
for i in 1 2 3; do cat ./dir/file_$i; done - открыть для чтения файлы из списка i
cat file{1, 2, 3} - открыть для чтения 3 файла
sudo chmod -R 777 /media/user/papka - дать полный доступ до директории. О правах подробнее ЗДЕСЬ(таблица)

mkfs.ext4 /dev/sda1 - создать ФС на разделе
tune2fs -l /dev/sda1 - просмотр суперблока ФС
•
mkdir /mnt/hello- создем точку монтирования
mount /dev/sdc1 /mnt/hello - монтируем раздел
df -h - просмотр смонтированных устройств

Увеличение ФС
1. fdisk /dev/sda- удаляем раздел командой d и создаем новый с увеличенным размером (+3G)
2. Выходим с сохранением w и при необходимости даем понять системе, что раздел изменился - командой partrobe.
Можно самостоятельно проверить, поняла ли система, в каталоге cat /proc/partitions
3. resize2fs /dev/sda1 - переоперделяем размер ФС.
e2fsck -f /dev/sda1 - проверка диска (при необходимости)
•
Уменьшение ФС
1. unmount /mnt/hello- размонтируем раздел
2. resize2fs /dev/sda1 1G - меняем размер ФС в меньшую сторону
3. mount /dev/sdc1/ mnt/hello - монтируем раздел
4. fdisk /dev/sda- удаляем раздел командой d и создаем новый с уменьшенным размером (+2G вместо трех, например)
partrobe - при необходимости
5. resize2fs /dev/sda1 - расширяем ФС (если до этого задавали с заделом меньше)
